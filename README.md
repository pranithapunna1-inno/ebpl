Morse Code based SoS System using ASK Signalling Scheme


This project is a wireless emergency alert system that transmits an SOS (... --- ...) message using ASK (Amplitude Shift Keying) communication. When the emergency button is pressed, the Arduino sends the Morse code through a 433 MHz RF transmitter. The receiver decodes the message and activates an LED and buzzer to indicate that an SOS signal has been received. The system is simple, low-cost, and suitable for emergency communication, safety applications, and learning wireless embedded system concepts.
